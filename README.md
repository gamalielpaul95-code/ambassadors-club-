A modern, real-time membership management system for **Ambassadors Club** at **Balili SDA Church**, Bunda, Mara - Tanzania. This application streamlines member registration, announcements, feedback management, and administrative controls with a beautiful, responsive interface.

## 🌟 Features

### 👤 Member Features
- **Secure Registration & Login** - JWT-based authentication
- **Personal Dashboard** - View personalized information and updates
- **Profile Management** - Upload profile picture, update personal details
- **View Announcements** - Read important club announcements in real-time
- **Submit Feedback** - Send suggestions and feedback to administrators
- **Real-time Notifications** - Instant updates on new announcements

### 👑 Admin Features
- **Member Management** - View, edit, and manage all club members
- **Position Assignment** - Assign roles (Ambassador, Leader, Coordinator, Member)
- **Member Status Control** - Activate/deactivate member accounts
- **Announcement Publishing** - Create and send announcements with priority levels
- **Feedback Management** - View and respond to member feedback
- **Real-time Dashboard** - Live updates on member activities
- **Complete Member Information** - Access all member details including registration date, last login, etc.

### 🚀 Technical Features
- **Real-time Updates** - Powered by Socket.io
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Image Upload** - Profile pictures via Cloudinary
- **Secure API** - Protected routes with JWT middleware
- **Database Indexing** - Optimized MongoDB queries
- **Error Handling** - Comprehensive error management
- **Form Validation** - Client and server-side validation
